## Get started

### Clone this repo
```
git clone git@github.com:nileorg/nile-prototype.git
```

### Init the submodules
```
git submodule init
```
```
git submodule update
```

### Run Docker!
```
sudo docker-compose up
```

## Use it

### Local instance
```
http://localhost:3334
```

### Node
